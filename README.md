# Notepad--

Desktop text editor created with tkinter in Python.

The project was created in order to learn GUI development, and application architecture.

## Features

- Open / Save / Save As
- Autosave
- Word & Character counter
- Line & Column indicator
- Find & Replace with navigation
- Light / Dark themes
- Custom themes saved to JSON
- Word wrap toggle
- Keyboard shortcuts
- Undo / Redo

## Installation & Run

1. Clone repository:
   git clone https://github.com/yourname/notepad.git

2. Navigate to project folder:
   cd notepad

3. Run:
   python3 main.py


## Project Structure

main.py
│
├── Notepad (Main application window)
├── FileManager (File operations + autosave)
├── EditorManager (Text operations)
├── PreferencesManager (Themes & settings)
└── Icons (UI assets)
